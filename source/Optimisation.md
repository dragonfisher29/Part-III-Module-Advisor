# Module profile

# **MATH3082 - Optimisation 2026-27**

  

  

## Module overview

The module provides an introduction to the theory and practice of optimization techniques. It covers linear programming as well as nonlinear programming. This module is suitable to those who want to apply computational optimization methods to their problems, which can arise from a variety of applied disciplines such as computer science and engineering.

## Aims

To be able to solve small sized problems by hand using Simplex Methods and Gradient Methods

  

## Learning Outcomes

  

|Reference|Learning Outcome|
|---|---|
|A1|Demonstrate knowledge and understanding of the basic ideas underlying optimization techniques|
|A2|Demonstrate knowledge and understanding of some of the most common standard optimization models and how they can be solved, including Simplex methods, gradient methods and Lagrangian function theory|
|A3|Understanding the basic optimization theory including optimality conditions and duality theory|
|B1|Develop mathematical optimization models for a range of practical problems|
|B2|Appreciate the power of using the mathematical approach to optimization problems relevant to engineering|

  

## Syllabus summary

1. Introduction to optimization modes including linear and nonlinear models 2. Graphical method for linear programming with two variables 3. Simplex methods (Phase I and Phase II methods, Dual simplex method) for linear programming 4. Duality theory and sensitivity analysis 5. Theorems of complementarity and the alternatives 6. Search methods (gradient methods) for nonlinear optimization 7. Lagrangian function theory 8. Practical use of software in solving linear programming
